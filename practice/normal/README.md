# 파이썬 기초 실습 문제

본 실습 예제들은 파이썬 기초 문법을 실무적 관점에서 익힐 수 있도록 설계되었습니다. 기초 연산부터 중첩 자료구조 핸들링까지 단계별로 구성되어 있습니다.

## 📊 문제 리스트 및 학습 목표

| 번호 | 주요 주제 | 학습 핵심 내용 | 난이도 |
| :--- | :--- | :--- | :---: |
| **Q1** | 평균값 산출 | 산술 연산 및 데이터 타입 변환 (`int`, `float`) | ★☆☆ |
| **Q2** | 문자열 처리 | 슬라이싱 및 반복문을 활용한 문자열 뒤집기 | ★☆☆ |
| **Q3** | 리스트 순회 | 반복문을 통한 리스트 원소 일괄 가공 (Mapping) | ★☆☆ |
| **Q4** | 성적 판독기 | 함수 정의, 다중 조건문, 조기 반환(`Early Return`) 패턴 | ★★☆ |
| **Q5** | 수치 연산 | 함수의 매개변수(`Parameter`) 활용 및 절댓값 로직 구현 | ★☆☆ |
| **Q6** | 데이터 대조 | 멤버십 연산자(`in`) 및 자료구조 비교를 통한 데이터 필터링 | ★★☆ |
| **Q7** | API 데이터 추출 | 중첩된 `Dict/List` 구조에서 특정 Key 값 수집 | ★★☆ |
| **Q8** | 도서 인기 순위 | `sort/sorted`와 `lambda`를 활용한 다중 조건 정렬 로직 | ★★★ |
| **Q9** | 도메인 정제 | 문자열 분리(`split`), 중복 제거(`set`) 및 정렬 | ★★☆ |
| **Q10** | 매출 집계 | 딕셔너리를 활용한 데이터 그룹화 및 누계(Aggregation) | ★★☆ |
| **Q11** | 2등 학생 찾기 | 정렬 및 중복 제거를 조합한 순위 데이터 추출 능력 | ★★★ |
| **Q12** | 결측치 처리 | `None` 값 탐지 및 통계적 수치를 활용한 데이터 보정(Imputation) | ★★★ |

---

## 🔍 문제별 상세 가이드

### [Step 1: 기초 논리 및 자료 구조]
*   **Q1~Q3**: 파이썬의 가장 기본적인 연산과 리스트/문자열을 다루는 법을 배웁니다. 데이터 분석의 전처리 과정에서 가장 빈번하게 발생하는 '형 변환'과 '일괄 수정'을 몸에 익힙니다.
*   **Q4~Q5**: 함수(Function)를 통해 로직을 구조화합니다. 특히 **Q4(성적 판독기)**는 실무에서 효율적인 코드를 짤 때 필수적인 '조기 반환' 패턴을 연습하는 핵심 예제입니다.

### [Step 2: 데이터 탐색 및 가공]
*   **Q6~Q7**: 전형적인 API 응답 데이터 형태(JSON 스타일)를 다룹니다. 복잡하게 얽힌 딕셔너리와 리스트 속에서 내가 원하는 정보만 쏙쏙 뽑아내는 '데이터 인덱싱' 감각을 기릅니다.
*   **Q8**: **도서 인기 순위** 정렬은 데이터 분석의 꽃인 '정렬'을 마스터하는 문제입니다. 여러 가지 기준(대출수, 연도 등)을 한 번에 적용하는 `lambda` 식의 위력을 경험합니다.

### [Step 3: 실무형 데이터 핸들링]
*   **Q9 (도메인 정제)**: 이메일 텍스트에서 도메인만 추출하거나 중복을 제거하는 등, 실제 로그 데이터를 정제할 때 사용하는 패턴을 학습합니다.
*   **Q10 (매출 집계)**: 딕셔너리의 키(Key)를 그룹화 도구로 사용하여 매출을 합산합니다. SQL의 `GROUP BY`나 Pandas의 `groupby` 동작 원리를 이해하는 기초가 됩니다.
*   **Q11 (2등 찾기)**: 단순히 최대값을 구하는 것을 넘어, 데이터의 등수를 매기고 특정 순위를 추출하는 복합 로직을 구현합니다.
*   **Q12 (결측치 처리)**: 데이터 분석 중 반드시 마주하게 되는 `None`(결측치)을 처리합니다. 유효한 데이터의 평균을 구하고 이를 빈칸에 채워 넣는 과정은 데이터 전처리의 표준 작업(Standard Imputation)입니다.
