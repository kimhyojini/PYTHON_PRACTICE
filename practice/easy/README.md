# 🐍 파이썬 기초 연습 문제 (초급)

이 디렉토리는 파이썬의 각 개념은 이해했으나, 실제로 코드를 작성하는 데 어려움을 느끼는 학생들을 위해 마련되었습니다.
문제를 단계별로 해결하며 로직을 구성하는 법을 익혀보세요.

## 📌 학습 가이드
1. **주석(Comment) 읽기**: 각 파일에는 해결해야 할 단계가 주석으로 설명되어 있습니다.
2. **순서대로 풀기**: E1부터 E8까지 난이도가 점진적으로 상승하므로 순서대로 푸는 것을 권장합니다.
3. **직접 코딩하기**: 눈으로 보는 것보다 직접 한 줄씩 타이핑하는 것이 실력 향상에 가장 좋습니다.

## 📂 문제 구성
- **Day 01 기초 (변수, 입출력, 조건문)**
  - `E1_hello_variable.py`: 사용자 입력과 기본 연산
  - `E2_if_logic.py`: 상황에 따른 조건 분기
- **Day 02 데이터 묶음 (리스트, 반복문)**
  - `E3_list_indexing.py`: 리스트 데이터 접근하기
  - `E4_for_loop_basics.py`: 반복문을 이용한 합계 구하기
- **Day 03 심화 자료구조 및 기초 알고리즘**
  - `E5_dict_lookup.py`: 딕셔너리를 활용한 정보 조회
  - `E6_algorithm_min_max.py`: 반복문과 조건문으로 최대/최소값 찾기
- **Day 04 함수 및 메서드**
  - `E7_function_basic.py`: 직접 함수 만들어보기
  - `E8_combined_logic.py`: 반복문과 조건문을 섞은 실전 로직
